# ⚡ Circuit Calc (Electrical Circuit Calculator)

An interactive web-based calculator designed to compute parameters for a series electrical circuit, including the internal resistance of the power source.

This project was successfully migrated from a **Windows Forms (C#)** desktop application to a modern **React/JS** stack to ensure cross-platform compatibility and browser accessibility.

## 🚀 Key Features
- **Accurate Calculations:** Computes current , external/internal voltage drops, and power balance.
- **Power Balance Verification:** Compares source power against total load power.
- **Responsive Design:** Fully optimized for both desktop and mobile devices using CSS Flexbox and Media Queries.
- **Interactive UI:** Features animated buttons and a clean, user-friendly layout.

## 🛠 Technologies Used
- **Frontend:** React.js / HTML5 / CSS3
- **Logic:** JavaScript (ES6+)
- **Deployment:** Vercel
